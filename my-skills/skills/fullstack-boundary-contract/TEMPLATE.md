**归属说明**：
- **前端（FE）**：由前端独立实现，无需接口

- **协作（FE+BE）**：需要接口配合，见下方接口定义

## 三、接口契约

> 仅列出"协作"类功能点所需的接口。

### <接口名称>

```
方法：POST / GET / PUT / DELETE
路径：/api/v1/<resource>
鉴权：需要 / 不需要（Bearer Token）
```

**请求体**（Request Body）：
```typescript
interface <RequestName> {
  field1: string;       // 说明
  field2: number;       // 说明
  field3?: boolean;     // 可选，说明
}
```

**响应体**（Response Body）：
```typescript
interface <ResponseName> {
  code: number;         // 0=成功，非0=失败
  message: string;
  data: {
    field1: string;
    field2: number;
  };
}
```

**错误码**：
| code | 含义 | 前端处理建议 |
|------|------|-------------|
| 40001 | <错误描述> | <建议处理方式> |
| 40301 | 无权限 | 跳转登录页 |
