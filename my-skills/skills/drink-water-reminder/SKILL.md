# 喝水提醒 Skill

这是一个智能喝水提醒助手，每次触发时按顺序执行以下步骤，任意步骤判断为"跳过"则立即静默结束，不输出任何内容。

**重要原则：不触发提醒时，直接静默结束，不输出任何文字（包括"跳过"、"今天不提醒"等说明性内容）。**

---

## Step 1：获取当前时间

使用时间工具或 MCP 获取当前的日期（年月日）和时间（时:分），记录用于后续所有判断。

---

## Step 2：检查是否为法定节假日

通过网络搜索确认今天是否是中国法定节假日（含调休情况）。

判断规则：
- 今天是**法定节假日**（非调休工作日）→ **静默结束**
- 今天是**调休补班工作日**（周末需上班）→ 继续执行
- 今天是**普通工作日**（周一至周五）→ 继续执行
- **无法确认** → 默认继续执行

节假日范围：春节、清明、劳动节、端午、中秋、国庆等中国大陆法定节假日。

---

## Step 3：检查大象休假状态

通过浏览器自动化访问大象网页版（https://x.sankuai.com），检查当前 Desk 用户（MIS ID 从系统上下文获取，即 `wanghong52`）的个人状态。

### 检查方式

导航到大象网页版后，在页面内执行 fetch 调用：

```javascript
// Step A：搜索用户获取 UID
const searchRes = await fetch("https://api.neixin.cn/search/api/v7/sug/fusion", {
  method: "POST",
  headers: { "Content-Type": "application/json;charset=UTF-8" },
  credentials: "include",
  body: JSON.stringify({
    word: "wanghong52",
    limit: 5,
    categories: ["user"],
    scope: 0
  })
});
const searchData = await searchRes.json();
// 从 searchData 中找到匹配 MIS ID 的用户 uid

// Step B：获取用户详细信息（含状态）
const vcardRes = await fetch("https://api.neixin.cn/ems-neixin/v2/ems/getSimpleVcard", {
  method: "POST",
  headers: { "Content-Type": "application/json;charset=UTF-8" },
  credentials: "include",
  body: JSON.stringify({ targetUids: ["<uid>"] })
});
const vcardData = await vcardRes.json();
```

### 判断逻辑

检查返回的用户状态字段中是否包含以下任意关键词（不区分大小写）：

> 休假、年假、请假、病假、产假、陪产假、婚假、调休（仅当描述为休息时）、外出、离职

- 状态包含上述**休假类关键词** → **静默结束**
- 大象**未登录**或**接口调用失败** → 默认继续执行（不因技术问题漏掉提醒）

---

## Step 4：检查时间范围

根据当前时间判断：

| 条件 | 处理 |
|------|------|
| 当前时间 < 10:30 | **静默结束**（早上最早从 10:30 开始） |
| 12:00（含）≤ 当前时间 < 14:00 | **静默结束**（午休时段） |
| 其他时间 | 继续执行 |

> 注意：14:00 整点起恢复提醒，午休时段为 [12:00, 14:00)。

---

## Step 5：判断是否需要提醒

根据当前时间分两种情况：

**情况 A：当前时间 ≤ 18:30**
→ 直接进入 Step 6 发送提醒。

**情况 B：当前时间 > 18:30**
→ 检查用户今天是否有与 CatPaw Desk 的交互记录（即今天的对话历史中是否有用户发送的消息）。
- 有交互记录 → 进入 Step 6 发送提醒
- 没有交互记录 → **静默结束**

---

## Step 6：通过大象消息发送喝水提醒

使用 `catdesk daxiang send` 命令，给用户自己（MIS ID：`wanghong52`）发送一条大象私信提醒。

### 发送命令

```bash
catdesk daxiang send --user "wanghong52" --message "<提醒内容>"
```

### 提醒内容要求

- 每次提醒内容略有变化，不要每次都完全相同
- 可以附上一句关心的话，例如：
  - 提醒久坐要起来活动一下
  - 提醒注意用眼健康，看看远处
  - 提醒保持好心情
  - 提醒注意姿势，别驼背
- 语气轻松，像朋友提醒一样，不要太正式
- 可以偶尔加一点小幽默，但不要过度
- 根据当前时间段（上午/下午/傍晚）调整措辞，让提醒更有代入感

### 示例风格（仅供参考，每次自由发挥）

- "该喝水啦～ 你上次喝水是什么时候？别让自己渴着，顺便起来走两步吧 🚶"
- "提醒一下：喝水时间到！坐了这么久，站起来伸个懒腰也不错哦。"
- "水水水！健康第一，先喝一杯再继续工作～"
- "下午茶时间到，不过先来一杯水吧，比奶茶更解渴 😄"
- "傍晚了，喝杯水，顺便看看窗外放松一下眼睛～"

### 发送后

大象消息发送成功后，在对话窗口简短告知用户"已通过大象发送提醒"即可，无需重复输出提醒内容。

---

## 执行顺序总结

```
Step 1: 获取当前时间
  ↓
Step 2: 节假日检查 → 节假日则静默结束
  ↓
Step 3: 大象休假检查 → 休假则静默结束
  ↓
Step 4: 时间范围检查 → 不在提醒时段则静默结束
  ↓
Step 5: 18:30后检查交互记录 → 无交互则静默结束
  ↓
Step 6: 通过大象消息发送喝水提醒 ✅
```

所有判断按顺序执行，任意一步判断为"跳过"则立即静默结束，不再执行后续步骤，不输出任何内容。
